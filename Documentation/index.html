About Cinagro


Cinagro – new unique, modern, with elegant curves and simple forms, fully responsive HTML theme, based on Bootstrap 3.3.6 created using HTML and LESS.

Install Theme


Important!


In order to get JS loading data you need to install localhost server or your computer must have internet access. To install the localhost server You can use WAMP. Then you need to:

  • Copy your files into *wamp/www
  • Run WAMP. This will start your localhost server.
  • Open browser and in URL type localhost/Cinagro (or other path)

Folder Structure

  • Cinagro
    • HTML Files
    • assets
      • audio - Audio File
      • css - CSS style sheets
      • fonts - Font Awesome font
      • images - Images
      • js - JS script files
    • documentation - This documentation
